📖 How to Use VoiceOver Studio
🚀 Quick Start Guide
1
Settings tab లో Groq API key add చేయండి — console.groq.com లో free గా తీసుకోవచ్చు
2
Create tab లో వెళ్ళి video upload చేయండి — gallery నుండి MP4, MOV select చేయండి
3
Video topic type చేయండి — "Craftland gameplay", "travel vlog" లాంటివి
4
👆 TAP ME button tap చేయండి — fast tap చేస్తే Energetic style, slow చేస్తే Calm style auto detect అవుతుంది!
5
Language select చేయండి — English లేదా Hindi
6
🚀 Generate Voiceover button నొక్కండి — AI script రాస్తుంది, voice add అవుతుంది, music generate అవుతుంది
7
⬇️ Download చేయండి — final video మీ phone లో save అవుతుంది
👆 Style Detection ఎలా పని చేస్తుంది?
Fast taps (5+ in 3 sec) → ⚡ Energetic style
Medium taps (3-4 in 3 sec) → 😊 Friendly style
Slow taps (2 in 3 sec) → 🎭 Dramatic style
Very slow (1 tap) → 😌 Calm style

లేదా manually style pills tap చేసి select చేయవచ్చు!
🎵 Music గురించి
Music మీరు select చేసిన style బట్టి automatically generate అవుతుంది:

⚡ Energetic → Fast, upbeat rhythm
😌 Calm → Soft, peaceful melody
🎭 Dramatic → Deep, cinematic sound
😊 Friendly → Happy, bright tones

100% copyright-free — browser లోనే generate అవుతుంది!
❓ Voice రావడం లేదా?
• Chrome browser use చేయండి (best support)
• Phone silent mode off చేయండి
• Volume max చేయండి
• Downloaded video లో sound check చేయండి
• iOS లో Safari కంటే Chrome better గా work చేస్తుంది
