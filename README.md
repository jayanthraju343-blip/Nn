✨ Features
📹 Video Upload — MP4, MOV, AVI support (up to 100MB)
🤖 AI Script Generation — Claude AI analyzes your video frames and writes a voiceover script
🗣️ Voice Languages — English 🇺🇸 and Hindi 🇮🇳
🎭 Voice Styles — Energetic, Calm, Dramatic, Friendly
🎵 Auto Background Music — Copyright-free music generated in browser (Chill Lofi, Epic, Happy, Dramatic, Gaming)
📱 Mobile Friendly — Works on phones and tablets
⬇️ Download — Final video with voice + music merged
🚀 How to Use
Option 1: GitHub Pages (Recommended)
Fork this repo
Go to Settings → Pages
Set source to main branch
Your app will be live at https://yourusername.github.io/voiceover-studio/
Option 2: Open Locally
Just open index.html in Chrome or Edge browser — no server needed!
🔑 Setup
You need an Anthropic API Key for AI script generation:
Go to console.anthropic.com
Create an account (free tier available)
Generate an API key
Paste it in the app when prompted
🔒 Your API key is never stored — it stays only in your browser session
